Dans ce jeu, votre but est de trouver, en 60 secondes, la pyramide dorée dans le labyrinthe. Vous obtiendrez des points pour chaque seconde restante.

Il y a un total de 10 niveaux à franchir pour terminer le jeu.

Après les 60 secondes, vous recommencerez le niveau si vous n'avez pas trouvé la pyramide.

Les murs de cèdres peuvent s'ouvrir avec la touche     espace     lorsque vous les regardez directement.

À tout moment, vous pouvez utiliser la vue aérienne en appuyant sur la touche Page Up. Appuyez sur la touche Page Down pour retourner où vous vous trouviez.

Mouvements : ← et → pour tourner sur place, ↑ et ↓ pour avancer ou reculer.

TRICHE : Appuyez simultanément sur les touches
CTRL + SHIFT +     espace    
pour faire apparaître la pyramide dans la vue aérienne.

ATTENTION!
Recommencer le niveau coûte 200 points et vous commencez avec 300.
Si vous n'avez plus de point, ce sera GAME OVER
Utiliser un ouvreur de mur coûte 50 points et vous en avez un nombre limité pour chaque niveau.
Vous perdrez 10 points par secondes passées dans la vue aérienne.
